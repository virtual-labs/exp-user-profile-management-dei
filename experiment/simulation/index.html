🚀 Simulation Steps

  1. Core Deployment: Use terminal or 🚀 One-Click Deploy to start all network functions.
  2. gNB Setup: Select gNB, configure its IP/Port, and deploy it to bridge UEs and CORE.
  3. UDR Verification: Select UDR and click "Show Subscriber Info" to verify registered UE data.
  4. Subscription Edit: Update UE parameters like DNN in UDR and save changes.
  5. UE Activation: Select UE, enter matching configuration, and start to verify 5G connectivity.
